# Keblo

Keblo is an experimental local AI platform exploring transparent, explainable, and modular persistent memory for language models.

## Features

- Local AI execution
- Persistent memory
- Explainable memory retrieval
- Memory graph visualization
- Modular architecture

## Built With

- GPT-5.6
- Codex
- Node.js
- JavaScript
- Ollama
- Qwen
- Redis
- Qdrant

## Codex Usage

Codex was used during OpenAI Build Week to implement, refactor, debug, review and validate multiple components of the project.

## GPT-5.6 Usage

GPT-5.6 was used to design architecture, review implementations, prepare documentation, and support development decisions.

## Status

This repository contains the Build Week prototype of Keblo and ongoing research on transparent persistent memory for local AI systems.
